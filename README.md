# 期末專案目標

結合這學期所學製作一套完整的系統，開發一套能辨識道路場景的智能系統。

以Jetbot自走車為基礎 (道路跟隨)。

使用路標偵測辨識交通指示牌並執行動作。

辨識不同速限的交通指示牌並讓Jetbot進行加速與減速。

# Final Project

* **專案項目：自走車**
* **讓 Jetbot 進行行駛並完成賽道。(30%)**
* 每次輪胎壓線扣 1 分。
* 輪胎每次超出車道一次扣 3 分，若車子完全脫離車道，則從脫離處重來。

* **讓 Jetbot 能夠偵測交通指示牌執行動作。(40%)**
* **當心行人**：偵測到此號誌後，減速後再繼續行駛。(7.5%)
* **道路封閉**：偵測到此號誌時停止 Jetbot，停止位置不得超過交通號誌。(7.5%)
* **停車再開**：偵測到此號誌時，於原地停止2秒後再繼續行駛。(7.5%)
* **鐵路平交道**：偵測到此號誌時，於原地停止5秒後再繼續行駛。(7.5%)
* 遇到假路標牌不該做動作，做一次少 2 分。(10%)

* **可嘗試次數：3 次**

* **撰寫操作過程學習心得並以小組報告(PDF)、實作Code、個人報告(PDF)。(30%)**
* **於 115/06/24 課堂上進行實體DEMO，視同期末考。**
